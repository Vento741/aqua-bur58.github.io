# ПЛАН SEO-ОПТИМИЗАЦИИ САЙТА АКВА-БУР58

## 1. ТЕХНИЧЕСКИЕ АСПЕКТЫ ОПТИМИЗАЦИИ

### 1.1. Метатеги и структурированные данные
- **Улучшение meta-description**
  - Оптимизировать description на всех страницах услуг с включением ключевых запросов и призывов к действию
  - Ограничить длину до 150-160 символов для корректного отображения в поисковой выдаче
  
- **Оптимизация title-тегов**
  - Включить ключевые фразы в начало title
  - Создать уникальные заголовки для каждой страницы
  - Ограничить длину до 60-70 символов
  
- **Микроразметка Schema.org**
  - Внедрить микроразметку LocalBusiness для улучшения представления в поисковой выдаче
  - Добавить разметку для адреса, телефонов, часов работы
  - Внедрить микроразметку для отзывов (Schema.org/Review)
  - Добавить разметку для услуг (Schema.org/Service)
  
- **Open Graph и Twitter Cards**
  - Дополнить и оптимизировать существующие Open Graph теги
  - Добавить Twitter Cards для лучшего отображения при шеринге в социальных сетях

### 1.2. URL-структура и навигация
- **Оптимизация URL-адресов**
  - Проверить все URL на использование понятных для пользователя и поисковиков адресов
  - Использовать транслитерацию для русских названий в URL
  
- **Хлебные крошки**
  - Оптимизировать существующие хлебные крошки на странице
  - Добавить микроразметку для хлебных крошек (Schema.org/BreadcrumbList)

### 1.3. Скорость загрузки сайта
- **Оптимизация изображений**
  - Конвертировать все изображения в формат WebP с сохранением PNG/JPG для старых браузеров
  - Установить правильные размеры изображений (width и height) для уменьшения CLS
  - Внедрить lazy loading для всех изображений за пределами первого экрана
  
- **Минификация и оптимизация кода**
  - Минифицировать CSS и JavaScript файлы
  - Объединить мелкие CSS и JS файлы для уменьшения количества HTTP-запросов
  - Использовать асинхронную загрузку для некритичных скриптов

- **Кеширование и предзагрузка**
  - Оптимизировать настройки кеширования в .htaccess
  - Добавить предзагрузку (preload) для критических ресурсов
  - Реализовать отложенную загрузку нативных JS скриптов

### 1.4. Мобильная оптимизация
- **Адаптивная верстка**
  - Проверить и улучшить отображение на всех типах устройств
  - Оптимизировать скорость загрузки на мобильных устройствах
  - Убедиться, что все элементы управления имеют достаточный размер для тач-устройств

- **Core Web Vitals**
  - Улучшить показатели LCP (Largest Contentful Paint)
  - Оптимизировать FID (First Input Delay)
  - Минимизировать CLS (Cumulative Layout Shift)

### 1.5. XML-карта сайта и файл robots.txt
- **Оптимизация sitemap.xml**
  - Обновить даты последнего изменения (lastmod) на актуальные
  - Проверить все URL на работоспособность
  - Добавить изображения в sitemap (image:loc)
  
- **Улучшение robots.txt**
  - Проверить корректность директив
  - Добавить инструкции для поисковых ботов Яндекса и Google

## 2. КОНТЕНТ-СТРАТЕГИЯ

### 2.1. Оптимизация существующего контента
- **Заголовки H1-H6**
  - Улучшить структуру заголовков для лучшей иерархии контента
  - Включить ключевые слова в заголовки разных уровней
  - Убедиться, что на каждой странице присутствует только один H1

- **LSI-ключи**
  - Расширить семантическое ядро за счет LSI-ключей
  - Внедрить тематические ключевые слова в контент
  
- **Оптимизация текстов**
  - Дополнить тексты на страницах услуг, добавив больше информации о процессе работы
  - Увеличить уникальность и экспертность текстов
  - Добавить региональные маркеры (Пенза, Пензенская область и т.д.)
  
- **Внутренняя перелинковка**
  - Создать более разветвленную структуру внутренних ссылок
  - Использовать релевантные анкоры для внутренних ссылок
  - Реализовать блок "Смотрите также" на страницах услуг

### 2.2. Создание нового контента
- **Добавление блога/статей**
  - Создать раздел блога с полезными статьями о водоснабжении
  - Темы для статей:
    - "Как выбрать тип скважины для вашего участка"
    - "5 распространенных проблем с водой из скважины и их решение"
    - "Сезонное обслуживание скважин: что нужно знать"
    - "Сравнение типов скважин: преимущества и недостатки"
  
- **FAQ-раздел**
  - Расширить существующий FAQ более детальными ответами
  - Добавить микроразметку FAQ (Schema.org/FAQPage)
  
- **Кейсы выполненных работ**
  - Создать раздел с примерами выполненных проектов
  - Добавить фото "до и после", описания проблем и их решений
  - Включить отзывы клиентов к каждому кейсу

### 2.3. Региональная оптимизация
- **Локальный SEO**
  - Создать отдельные страницы для наиболее важных районов Пензенской области
  - Добавить интерактивную карту обслуживаемых районов
  - Внедрить региональные ключевые слова в контент

- **Google My Business и Яндекс.Справочник**
  - Создать и оптимизировать профили в сервисах Google Мой бизнес и Яндекс.Справочник
  - Добавить актуальные фотографии объектов и процесса работы
  - Настроить сбор и модерацию отзывов клиентов
